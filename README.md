# amuryx-frontend-react-task-one

##  Frontend React Internship Task

This project is a simple **Product List application** built using **React** as part of the Amuryx frontend internship evaluation.

The main goal of this task is to demonstrate **problem-solving skills**, clean component structure, and basic frontend logic rather than complex features.

---

## Features

- Display a list of products
- Search products by name (case-insensitive)
- Sort products by price:
  - Low to High
  - High to Low
- Responsive UI (mobile, tablet, desktop)
- Clean and simple layout using CSS Grid
- Prices displayed in **Sri Lankan Rupees (LKR)**

---

##  Technologies Used

- React (Functional Components)
- React Hooks (`useState`)
- JavaScript (ES6)
- CSS (Grid & Responsive Design)
